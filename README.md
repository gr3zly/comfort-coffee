# Кофейня «Уют» – сайт-визитка

Современный одностраничный сайт кофейни с богатым интерактивом. Учебный проект по HTML, CSS и JavaScript.

## Файлы

- `index.html` – разметка страницы
- `style.css` – оформление и анимации
- `script.js` – вся интерактивная логика

## Реализованный функционал

### HTML
- Шапка (`header`) с логотипом и навигацией (О нас, Меню, Фото)
- Секция Hero с приветствием и кнопкой «Смотреть меню»
- Секция «О нас» с описанием и режимом работы
- Секция «Меню» с вкладками (Кофе / Еда)
- Секция «Галерея» с фотографиями блюд и атмосферы
- Секция «Оставить отзыв» с формой обратной связи
- Подвал (`footer`) с контактной информацией (адрес, телефон, почта)

### CSS
- **Адаптивный дизайн** – корректное отображение на всех устройствах (планшеты, мобильные телефоны)
- **Тёплая цветовая гамма** – бежевые, кофейные, оранжевые оттенки
- **Sticky-шапка** – навигация всегда видна при прокрутке
- **Hover-эффекты** – кнопки, карточки галереи, пункты меню реагируют на наведение
- **Анимации**:
  - Плавное увеличение фотографий при наведении
  - Пульсация при подсветке выбранного блюда
  - Плавное появление уведомлений
- **Сетка галереи** – адаптивная grid-сетка с квадратными карточками
- **Уменьшенные фотографии** – для латте, чизкейка, брауни и капучино размер изображений специально уменьшен для эстетики

### JavaScript

#### Навигация и прокрутка
- Плавная прокрутка к секциям при клике на пункты меню в шапке
- Плавная прокрутка к меню при клике на кнопку «Смотреть меню»
- Учёт высоты sticky-шапки при прокрутке

#### Меню с вкладками
- Переключение между вкладками «Кофе» и «Еда»
- Активная вкладка визуально выделяется

#### Интерактивное меню → галерея
- **Клик по любому блюду в меню** автоматически:
1. Переключает на нужную вкладку (Кофе или Еда), если требуется
2. Плавно прокручивает страницу к фотографии этого блюда в галерее
3. Подсвечивает фотографию анимированной пульсацией на 2 секунды
- Полное соответствие между блюдами и их фотографиями

#### Форма обратной связи
- Валидация полей (имя и отзыв не могут быть пустыми)
- Отображение ошибок под каждым полем
- Всплывающее уведомление (с анимацией):
  - При успешной отправке – зелёное уведомление с благодарностью
  - При ошибке – красное уведомление с пояснением
- Автоматическое очищение формы после успешной отправки
- Уведомление автоматически исчезает через 3 секунды

## Как запустить

Открыть файл `index.html` в любом современном браузере. Интернет не требуется (все стили и скрипты встроены в локальные файлы).

## Технологии

- HTML5
- CSS3 (Flexbox, Grid, анимации, медиазапросы)
- JavaScript (ES6)
